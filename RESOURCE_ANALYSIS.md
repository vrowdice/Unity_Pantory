# 자원(Resource) 상황 분석 및 설명

## 📊 ResourceType 매핑
- **0**: raw (원자재)
- **1**: metal (금속)
- **2**: wood (나무)
- **3**: tool (도구)
- **4**: weapon (무기)
- **5**: furniture (가구)
- **6**: clothing (옷)
- **7**: component (부품)
- **8**: electronics (전자제품)
- **9**: vehicle (차량)
